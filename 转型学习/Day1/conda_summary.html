🛠️ Conda 环境管理命令总结

为大语言模型（LLM）项目创建独立的 Python 开发环境

逐条命令说明
①
conda --version
查看当前安装的 Conda 版本号，确认 Conda 是否已安装及版本信息。
②
conda create -n llm python=3.11 -y
创建一个名为 llm 的虚拟环境，指定 Python 3.11 版本。
-n llm 表示环境名称，-y 表示自动确认跳过询问。
③
conda activate llm
激活（进入）llm 环境。激活后终端会显示 (llm) 前缀，
此后安装的包和运行的代码都隔离在此环境中。
④
python --version
验证当前激活环境中的 Python 版本，应显示 Python 3.11.x。
步骤总览
步骤 命令 目的
① conda --version 确认 Conda 正常可用
② conda create -n llm python=3.11 -y 创建隔离环境（Python 3.11）
③ conda activate llm 进入该虚拟环境
④ python --version 验证环境中的 Python 版本正确

🎯 核心目标

为 大语言模型（LLM） 相关项目创建一个独立、干净的 Python 3.11 开发环境。
使用虚拟环境可以隔离不同项目的依赖，避免包版本冲突，保证开发环境的可复现性。
