核心差异（必须记住）：

• Python 用缩进代替大括号 → 缩进错误 = 语法错误

• Python 列表推导式：[x*2 for x in range(10) if x > 5]

💻任务 6：第一个 Python 程序（城市温度筛选）实战45 分钟Trae

编程任务：

在 Trae 创建项目 llm-learning-journey，新建 day1/day1_basics.py：

# day1_basics.py
# 1. 定义字典存5个城市温度
cities = {
    "北京": 28, "上海": 25, "深圳": 32,
    "成都": 19, "哈尔滨": 12
}

# 2. 列表推导式筛选温度 > 20°C
hot_cities = [(city, temp) for city, temp in cities.items() if temp > 20]

# 3. f-string 格式化输出
print(f"温度高于20°C的城市共 {len(hot_cities)} 个：")
for city, temp in hot_cities:
    print(f"  {city}: {temp}°C")

# 4. 进阶：用 sorted 按温度降序
hot_cities_sorted = sorted(hot_cities, key=lambda x: x[1], reverse=True)
print("\n按温度降序排列：")
for city, temp in hot_cities_sorted:
    print(f"  {city}: {temp}°C  {'🔥' * (temp // 10)}")

运行：在 Trae 中按 F5 或终端 python day1/day1_basics.py

第一个完整 Python 程序，巩固字典、列表推导式、f-string、Lambda、排序 5 个核心知识点。
能看到输出 4 个城市（北京/上海/深圳/成都），且按温度降序排列。
💻任务 7：第一次调用大模型 API实战45 分钟Trae

操作步骤：

1. 创建 day1/.env 文件（注意加到 .gitignore）：

2. 创建 day1/.gitignore：

__pycache__/
*.pyc
.venv/

3. 创建 day1/day1_first_api.py：

import os
from dotenv import load_dotenv
from openai import OpenAI

load_dotenv()

client = OpenAI(
    api_key=os.getenv("API_KEY"),
    base_url=os.getenv("API_BASE")
)

# 同步调用
response = client.chat.completions.create(
    model=os.getenv("MODEL"),
    messages=[
        {"role": "system", "content": "你是一个有帮助的AI助手。"},
        {"role": "user", "content": "用一句话介绍什么是大语言模型。"}
    ],
)
print("【同步输出】", response.choices[0].message.content)
print("\n" + "=" * 50 + "\n")

# 流式调用
print("【流式输出】", end="")
stream = client.chat.completions.create(
    model=os.getenv("MODEL"),
    messages=[
        {"role": "user", "content": "用 3 个要点介绍 Transformer 架构。"}
    ],
    stream=True
)
for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
print()

4. 运行并观察流式效果
